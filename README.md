## 🛠️ Tech Stack & Architecture

*   **Framework:** [Next.js](https://nextjs.org/) (React)
*   **Styling:** [Tailwind CSS](https://tailwindcss.com/)
*   **Animations:** [GSAP](https://gsap.com/) (ScrollTrigger, Timelines)
*   **3D Elements:** [Three.js](https://threejs.org/)
*   **Data Management:** JSON-driven architecture for rapid content updates.

## 📂 Project Structure

This repository uses a clean, data-driven approach. The core application logic is separated from the profile content, allowing for instant updates without touching UI components.

```text
cinematic-portfolio-main/
├── app/                  # Next.js App Router and layout components
├── components/           # UI, 3D, and layout sections
├── data/                 
│   ├── profile.json      # Core data: Experience, Skills, Projects
│   └── content.json      # UI copy, labels, and section text
├── public/
│   ├── assets/           # Video backgrounds, avatars, and project images
│   └── Anirudh_Koppuravuri_Intern_Resume.pdf
├── styles/               # Global CSS and Tailwind configurations
└── lib/                  # Utility functions and GSAP configurations
